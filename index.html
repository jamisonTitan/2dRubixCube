Instructions

  • Click a side to select it
  • fc = rotate face clockwise
  • fcc = rotate face cunter clockwise
  • lu = turn left column up
  • ld = turn left column down
  • ru = turn right column up
  • rd = turn right column down
  • bl = turn bottom row left
  • br = turn bottom row right
  • tl = turn top row left
  • tr =trun top row right
